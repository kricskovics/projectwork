# User Stories

User Story 1: Modify Language<br>
Description: As a user I want to modify the language on the site.

User Story 2: Plan a Trip<br>
Description: As a user I want to plan a trip and display all results.

User Story 3: Search for a Route<br>
Description: As a user I want to search a route by its number and see the list of the stops and the route.

User Story 4: Search for an Address<br>
Description: As a user I want to search for all available routes and means of transportation from a specific address and display all results.

User Story 5: Report a Bug<br>
Description: As a user I want to be able to report bugs on the website.
